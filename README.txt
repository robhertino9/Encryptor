In order to run this application you will need the following

Python latest version (3.11.3 was used for development) - can be installed on www.python.org/downloads
Install Cryptography module (40.0.2 used for development) -  Installation guide https://cryptography.io/en/latest/installation/
Install PIL module (9.5.0 Used for development) - Installation guide https://pillow.readthedocs.io/en/latest/installation.html
